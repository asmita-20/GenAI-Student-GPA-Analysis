GenAI Impact on Student GPA | Power BI + Python
Project Overview
Analyzed a dataset of 200 students to find if Generative AI usage hours impact GPA
 Key Findings
1 No Correlation Pearson Correlation = -0.0057 between AI Hours and GPA.
2.Tool Usage :70% + students use only 1-2 AI tools.
3. GPA Distribution: Normally distributed with avg 7.5
Tech Stack
Power BI: Data Modeling, DAX, Scatter, Bar, Histogram 
Python:Pandas, Seaborn for EDA & Verification
